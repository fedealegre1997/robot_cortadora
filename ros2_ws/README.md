# ros2_ws

Workspace ROS 2 del proyecto.

## Función

Aquí se compilan y ejecutan los paquetes ROS 2 del host.

## Contenido

- `src/`: código fuente de los paquetes ROS 2
- `build/`: archivos generados de compilación
- `install/`: instalación local del workspace
- `log/`: logs de compilación

## Uso

### Compilar
```bash
cd ~/robot_cortadora/ros2_ws
colcon build
