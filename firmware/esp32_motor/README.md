# esp32_motor

Firmware del ESP32 encargado del control de motores.

## Función

Recibe comandos de movimiento desde ROS 2 y publica estado de ejecución.

### Publica

- `active/`: proyecto ESP-IDF actual
- `versions/`: respaldos del código
- `notes/`: notas y observaciones
