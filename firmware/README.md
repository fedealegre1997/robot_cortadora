# firmware

Contiene los proyectos de firmware para los ESP32 del sistema.

## Subcarpetas

- `esp32_motor/`: firmware del ESP32 encargado de motores
- `esp32_sensor/`: firmware del ESP32 encargado de sensores

## Regla

Cada firmware tiene:
- `active/`: proyecto actual compilable
- `versions/`: versiones anteriores o respaldos
- `notes/`: notas técnicas y observaciones
