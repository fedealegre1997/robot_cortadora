# scripts common

Scripts comunes de entorno y ejecución del host ROS 2.

## Scripts

- `env.sh`: carga entorno del proyecto
- `build_ros2.sh`: compila el workspace ROS 2
- `run_agent.sh`: ejecuta micro-ROS Agent
- `run_brain.sh`: ejecuta el nodo principal
- `run_system.sh`: pensado para arrancar el sistema completo
