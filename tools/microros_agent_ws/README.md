# microros_agent_ws

Workspace auxiliar para construir y ejecutar el micro-ROS Agent en la PC Linux.

## Qué es

El micro-ROS Agent es el puente entre los ESP32 con micro-ROS y el sistema ROS 2 del host.

## Para qué sirve

Permite que los nodos micro-ROS de los ESP32 aparezcan en ROS 2 como topics y nodos normales.
